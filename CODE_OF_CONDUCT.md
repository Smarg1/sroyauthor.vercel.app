# Code of Conduct

## Purpose

This project maintains a professional environment focused on technical clarity, respect, and constructive collaboration.

## Expectations

- Treat all contributors with respect.
- Keep discussions focused on the code and the work.
- Provide explanations when suggesting changes or rejecting ideas.
- Avoid harassment, personal attacks, or derogatory behavior.
- Do not use this project for promotion, advertising, or unrelated activity.

## Enforcement

Violations may result in:

- Temporary interaction limits
- Removal from contributions

The project maintainer is responsible for applying these rules.

## Contact

For conduct-related concerns, contact the maintainer privately through GitHub’s reporting channels.
