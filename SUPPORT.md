# Support

This project does not provide general user support or troubleshooting channels.

## Bug Reports

For reproducible bugs:

- Open a GitHub issue.
- Include environment details, steps to reproduce, and expected vs. actual behavior.

## Security Issues

Do **not** create public issues for security problems.  
Use GitHub’s private vulnerability reporting system.

## Feature Requests

Feature requests may be submitted as issues. Approval depends on project scope and maintainer review.

## Availability

Support is provided on a best-effort basis.
